# Goals
This project aims to explore the use of PHP and Laravel with SSR principles using Blade Template.

## Commands 
To start the project
```php artisan serve```

## Docker commands
To build the project
```docker-compose build```

To run the project
```docker-compose up```

or

```docker-compose up -d```
